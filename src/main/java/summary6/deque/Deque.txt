addFirst(element) - добавление элемента в голову д/н очереди
addLast(element) - добавление элемента в хвост д/н очереди

E getFirst() - возвращает первый элемент (из головы) -элемент НЕ УДАЛЯЕТСЯ
E getLast() - возвращает последний элемент (из хвоста) -элемент НЕ УДАЛЯЕТСЯ

E pop() - возвращает ("вытаскивает") элемент из головы и удаляет из головы
push(E) - добавляет элемент в голову

E removeFirst() - возвращает ("вытаскивает") элемент из головы и удаляет его из коллекции
E removeLast() - возвращает ("вытаскивает") элемент из хвоста и удаляет его из коллекции

если элементов в очереди нет - то возвращается исключение IllegalStateException


-=======

E peekFirst() - возвращает элемент из головы без удаления из коллекции
E peekLast() - возвращает элемент из хвоста без удаления из коллекции

E pollFirst()
E pollLast() - возвращает элемент из головы / хвоста - удаляет его из коллекции

если элементов в очереди нет - то возвращается null!!!
