2. Расстановка кораблей первого игрока
    - запросить для каждого корабля по очереди
        - направление - горизонталь или вертикаль;
        - координаты начала
    - проверить правильность расстановки (не по диагонали, не может быть "кривой" корабль)
    - нет рядом соседних кораблей

3. Расстановка кораблей второго игрока

4. запуск битвы:
    - ход первого игрока
    - проверка результат выстрела
    - либо второй выстрел либо переход хода
    - тоже самое для второго игрока

5. Итог игры
