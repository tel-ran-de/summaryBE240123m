Нам нужно создать меню программы

1 сценарий:

создали класс START в котором есть метод start в котором перечисленны пункты меню,
например 1-5
далее мы запрашиваем ввод от пользователя
получаем значение
проверяем его корректность
в зависимости от выбора - вызываем тот или иной метод для выполнения


возникает вопрос - добавить новые пункты меню

что бы мы делали в таком случае?

1) взяли старый код
2) переделали его - добавили ноые пункты
3) оттестировали
4) запустили в работу

2 сценарий:

создаем класс который будет использовать интерфейс start
в котором будет реализован метод запуска конкретного действия
и создаем столько экземляров класса сколько нам нужно действий

наше меню будет универсальным! т е в нем не будет указано конкретное число пунктов

оно будет получать на вход массив данных, содержащее список пунктов и класс с ними ассоцтированный

в случае необходимости добавления новых пунктов мы просто в одно место, где формируется этот список
добавим новые пункты и все!
